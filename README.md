# iptv-best.ru

Русскоязычный сайт с обзорами и сравнением IPTV-провайдеров.

## Стек

- Next.js 15 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## Запуск

```bash
npm install
npm run dev
```

Сайт будет доступен по адресу `http://localhost:3000`. Для продакшена на домене `iptv-best.ru` рекомендуем деплой на Vercel или аналогичный хостинг.

## Контент

Текстовые описания провайдеров берутся из открытых источников и **переписываются своими словами** (без прямого копирования). Следите за тем, чтобы итоговый текст был уникальным и не нарушал авторские права.
